The user wants to know the startup sequence when they come back to work on the project. Let me think about what they need to do each time they start a new work session.

Quy trình mở lại dự án
1 lệnh duy nhất để start tất cả:

cd ~/Desktop/ATS-DW
make up
Đợi ~30 giây rồi verify:


make ps
Tất cả phải hiện healthy / running.

Các URL sau khi up:
Service	URL	Login
Airflow	http://localhost:8080	admin / password trong .env
MinIO	http://localhost:9001	MINIO_ACCESS_KEY / MINIO_SECRET_KEY
Metabase	http://localhost:3000	setup lần đầu
PostgreSQL	localhost:5434	dùng DBeaver/TablePlus
Khi kết thúc session:

make down
Lưu ý quan trọng:
PostgreSQL data, MinIO data, Airflow logs đều được lưu trong Docker volumes — tắt máy/container không mất data
Chỉ mất data nếu chạy docker volume rm hoặc docker-compose down -v (flag -v xóa volumes)
